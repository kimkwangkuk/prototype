# Todo App - 프로토타입 프로젝트

## 프로젝트 철학

**이 프로젝트는 프로덕션이 아닌 프로토타입입니다.**


### 개발 원칙
- ⚡ **빠른 구현**: 복잡한 아키텍처보다 동작하는 코드
- 💰 **경제적 개발**: 최소한의 의존성, 빌드 시간 최소화
- 🔄 **빠른 반복**: 쉽게 만들고, 쉽게 버리고, 쉽게 다시 만들기
- 🎯 **현재 요구사항**: 확장성보다 지금 필요한 기능에 집중

---


## 에이전트 활용

- **frontend-developer**: 새 기능 구현 및 수정
- **qa-engineer**: 코드 분석 및 엣지케이스 발견
